1. PRIVACY POLICY 
**Last Updated:** July 10, 2026
This privacy policy governs your use of the software application **Tool Pack** ("Application") for mobile devices, which
was created by the developer. 
A. Data Collection and Usage
The Application does not collect, transmit, store, or share any personal information, location data, or device identifiers. 
B. Device Permissions Required
To provide its utility features, the Application requests access to the following hardware components:
  - **Camera Access:** Used solely in real-time to estimate distances (Rangefinder) and capture high-speed frames for
slow-motion mechanics inspections (Inspection Camera).
  - **Microphone Access:** Used exclusively in real-time to analyze sound amplitude levels for decibel measurements
(Sound Meter).
  - **Physical Sensors (Accelerometer, Gyroscope, Magnetometer, Barometer, Light Sensor):** Used in real-time to
calculate angles, bubble alignment, magnetic fields, vibration severity, altitude shifts, and lighting intensity.
